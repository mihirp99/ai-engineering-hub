# AI Engineering Hub

Central hub for Cursor AI Engineering Team configuration, JARVIS assistant, project templates, and integration docs.

## Cloud Agent Setup

This repo is configured for **Build in Cloud** (Cursor Cloud Agents).

1. Push this repo to GitHub
2. Connect GitHub in [Cursor Dashboard → Integrations](https://cursor.com/dashboard/integrations)
3. Create a cloud environment at [Cloud Agents](https://cursor.com/dashboard/cloud-agents)
4. Open this folder in Cursor → Agent dropdown → **Cloud** → Build

## Structure

```
ai-engineering-hub/
├── docs/           Project documentation
├── template/       Greenfield project template
├── scripts/        Automation scripts
└── .cursor/        Cloud environment + rules
```

## JARVIS Commands

Type in Cursor chat: `/status`, `/tasks`, `/help`, `/roadmap`, `/deploy`
